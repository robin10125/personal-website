Some Work Projects

I designed an implemented a real time speech to text transcription pipeline, as well as as well as experiments with improving neural embedding based search/retrieval techniques..

I made a real time speech to text transcription program in Python using OpenAI's whisper models, combined with automatic speech recognition (ASR). I used a rolling audio window sliding forward at an abritrary frequency to capture patches of audio data, where were then condensed using ASR into only speaking parts, and then fed into a Whisper model for infernence.

This gave a series of semi overlapping text chunks, which were then recombined into a single coherent stream using approximate string matching algorithms, timestamp-aware word alignment.

I also designed diverse beam search retrieval augmented generation pipeline to pair with this, to generate a diverse range of information related to the real time transcription. This uses diversity metrics to augment beam search to ensure that each branch is realtively unique. Then each diverse branch is used for embedding based similarity search.

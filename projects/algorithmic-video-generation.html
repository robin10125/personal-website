Algorithmic Video Generation

I used coding agents, particularly claude code, to generate scripts that generate videos.

I used claude code to generate videos based off of a set of requirements, using ffmpeg and other video and image editing libraries. This involved a lot of trial and error, but was an interesting project testing the boundaries of coding agent capabilities.

I was making these videos for an event that I was running: Alchemy.

This project made clear the kinds of seemingly outlandish ideas that coding agents can execute on, beyond just coding apps. We can use these coding agents to abstract the whole process of app development, and operate at a new level of unexplored abstraction: what can you create if making programs was quick and easy?.

This project inspires me to try many other ideas I normally wouldn't pursue, like smart home automation, personal robotics, smartphone UX design, and many other cool ideas in the idea queue.

KL Divergence

June 11, 2026

I refreshed my understanding of KL divergence, focusing on forward KL, reverse KL and their applications and related concepts. In summary, KL divergence measures the difference between an estimated probability distribution and a real probability distribution. An example may be a model's predicted likelyhood of observing data x and the ground truth likelyhood.

The formula for forward KL divergence is:

DKL(P || Q) = 𝔼x∼P [log(P(x) / Q(x))] = ∑x P(x) log(P(x) / Q(x))

We can see that forward KL divergence penalizes the model (Q) for predicting low values of Q where the true value (P) is not low. At the extreme, if Q(x)=0, then if P(x)=/=0, then the KL divergence will explode to infinity. Alternatively, if Q(x) assigns high probablilty where P(x) is very low or zero, there is little to no penalty (low KL divergence), since each term is scaled by P(x). These two dynamics bias the model Q to model a wide distribution that encompasses all the values P could take, and often take the same mean value as P.

The formula for reverse KL divergence is

DKL(Q || P) = 𝔼x∼Q [log(Q(x) / P(x))] = ∑x Q(x) log(Q(x) / P(x))

We see that the reverse KL divergence scales each term by the value of Q(x), so that reverse KL divergence is not affected much by differences in distributions where Q(x) is small. Therefore reverse KL divergence is only really concerned with the accuracy of model predictions, and not whether these accurately model all of P. We see the value of reverse KL divergence explode if a large value of Q(x) is predicted where P(x) is small. When optimizing using this, false negatives are not penalized, however false positives are penalized heavily. This can produce a more narrow band of behaviour centering around the modal values of P.

Some more connections to ML

We can see that optimizing cross entropy is very similar to forward KL divergence. When training a model, the real distribution P doesn't change, only Q, and therefore optimzing KL divergence is equivalent to optimizing the -\sum_x{-P(x)log(Q(x)} term, which is exactly the cross entropy from P to Q.

We can also get the ELBO from reverse KL divergence, which comes from optimizing the reverse KL divergence between an approximate posterier q(z|x) and a true posterior p(z|x). We take our reverse KL divergence formula with the posterior P, and we get the equation D_{KL}(q(z|x) || p(z | x)) = log(p(x)) - E_{q}[log(p(x,z)-log(q(z|x))]. The second term is the (negative) ELBO. We can rearrange this to get log(p(x) = D_{KL}(...) + ELBO. Since KL divergence is always non negative, we get the inequality log(p(x)) >= ELBO, meaning the ELBO is a lower bound on the log evidence of p(x). More details on the ELBO deserve its own post.

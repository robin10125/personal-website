Pong Q Network

I trained a Q network that plays pong.

As an introduction to reinforcement learning, I trained a Q network that plays pong. The network takes as input the last few frames of the game of pong, and outputs an action for the pong bar (left or right). The Q network was trained against a competent but not optinmal pong engine. This was a simple and easy projec that introduced me to Q networks, RL theory, and Open AI's gym environment. I also set up a linux dual boot so that I could do this project, so it also introduced me to linux OS, which I continue to use today.
